<UnderDevelopment />

MongoDB support is planned for RustAuth but not shipped yet. Use a SQL adapter ([SQLx](/docs/adapters/sqlx), [Deadpool Postgres](/docs/adapters/deadpool-postgres), or [Tokio Postgres](/docs/adapters/tokio-postgres)) until the MongoDB adapter lands.

Track progress in the [migration matrix](/docs/MIGRATION_MATRIX.yaml) (`docs/adapters/mongo` → `under_development`).
