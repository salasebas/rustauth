---
title: Salesforce
description: Salesforce OAuth provider for RustAuth.
---

Register an OAuth application with Salesforce. Set the redirect URL to `{RUSTAUTH_BASE_URL}/callback/salesforce`.

## Configure RustAuth

```toml
[dependencies]
rustauth = { version = "0.1", features = ["social-providers"] }
```

```rust
use rustauth::prelude::*;
use rustauth::social_providers::providers;
use rustauth::social_providers::SocialProviderConfig;
use rustauth::social_providers::advanced::salesforce::{salesforce, SalesforceEnvironment, SalesforceOptions};

let auth = RustAuth::builder()
    .secret(std::env::var("RUSTAUTH_SECRET")?)
    .base_url(std::env::var("RUSTAUTH_BASE_URL")?)
    .social_provider(salesforce(SalesforceOptions {
    oauth: SocialProviderConfig::from_env("SALESFORCE")?.into_provider_options(),
    environment: SalesforceEnvironment::Sandbox,
    ..SalesforceOptions::default()
})?)
    .build()
    .await?;
```

Credentials: `SALESFORCE_CLIENT_ID`, `SALESFORCE_CLIENT_SECRET` (and any provider-specific env vars shown above).

## Sign in

```
GET {base_url}/sign-in/social?provider=salesforce
```

Callback: `GET {base_url}/callback/salesforce`.
