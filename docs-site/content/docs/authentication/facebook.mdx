---
title: Facebook
description: Facebook OAuth provider for RustAuth.
---

Register an OAuth application with Facebook. Set the redirect URL to `{RUSTAUTH_BASE_URL}/callback/facebook`.

## Configure RustAuth

```toml
[dependencies]
rustauth = { version = "0.1", features = ["social-providers"] }
```

```rust
use rustauth::prelude::*;
use rustauth::social_providers::providers;
use rustauth::social_providers::SocialProviderConfig;
use rustauth::social_providers::advanced::facebook::{facebook, FacebookOptions};

let auth = RustAuth::builder()
    .secret(std::env::var("RUSTAUTH_SECRET")?)
    .base_url(std::env::var("RUSTAUTH_BASE_URL")?)
    .social_provider(facebook(FacebookOptions {
    oauth: SocialProviderConfig::from_env("FACEBOOK")?.into_provider_options(),
    fields: vec!["email".into(), "name".into()],
    config_id: std::env::var("FACEBOOK_CONFIG_ID").ok(),
    ..FacebookOptions::default()
})?)
    .build()
    .await?;
```

Credentials: `FACEBOOK_CLIENT_ID`, `FACEBOOK_CLIENT_SECRET` (and any provider-specific env vars shown above).

## Sign in

```
GET {base_url}/sign-in/social?provider=facebook
```

Callback: `GET {base_url}/callback/facebook`.
