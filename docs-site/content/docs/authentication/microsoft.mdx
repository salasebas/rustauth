---
title: Microsoft Entra ID
description: Microsoft Entra ID OAuth provider for RustAuth.
---

Register an OAuth application with Microsoft Entra ID. Set the redirect URL to `{RUSTAUTH_BASE_URL}/callback/microsoft`.

## Configure RustAuth

```toml
[dependencies]
rustauth = { version = "0.1", features = ["social-providers"] }
```

```rust
use rustauth::prelude::*;
use rustauth::social_providers::providers;
use rustauth::social_providers::SocialProviderConfig;
use rustauth::social_providers::advanced::microsoft_entra_id::{microsoft_entra_id, MicrosoftEntraIdOptions};

let auth = RustAuth::builder()
    .secret(std::env::var("RUSTAUTH_SECRET")?)
    .base_url(std::env::var("RUSTAUTH_BASE_URL")?)
    .social_provider(microsoft_entra_id(MicrosoftEntraIdOptions {
    oauth: SocialProviderConfig::from_env("MICROSOFT")?.into_provider_options(),
    tenant_id: std::env::var("MICROSOFT_TENANT_ID").ok(),
    ..MicrosoftEntraIdOptions::default()
})?)
    .build()
    .await?;
```

Credentials: `MICROSOFT_CLIENT_ID`, `MICROSOFT_CLIENT_SECRET` (and any provider-specific env vars shown above).

## Sign in

```
GET {base_url}/sign-in/social?provider=microsoft
```

Callback: `GET {base_url}/callback/microsoft`.
