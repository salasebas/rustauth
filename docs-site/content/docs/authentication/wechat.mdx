---
title: WeChat
description: WeChat OAuth provider for RustAuth.
---

Register an OAuth application with WeChat. Set the redirect URL to `{RUSTAUTH_BASE_URL}/callback/wechat`.

## Configure RustAuth

```toml
[dependencies]
rustauth = { version = "0.1", features = ["social-providers"] }
```

```rust
use rustauth::prelude::*;
use rustauth::social_providers::providers;
use rustauth::social_providers::SocialProviderConfig;
use rustauth::social_providers::advanced::wechat::{WeChatLang, WeChatProvider, WeChatProviderOptions};

let auth = RustAuth::builder()
    .secret(std::env::var("RUSTAUTH_SECRET")?)
    .base_url(std::env::var("RUSTAUTH_BASE_URL")?)
    .social_provider(WeChatProvider::new(WeChatProviderOptions {
    oauth: SocialProviderConfig::from_env("WECHAT")?.into_provider_options(),
    lang: Some(WeChatLang::En),
})?)
    .build()
    .await?;
```

Credentials: `WECHAT_CLIENT_ID`, `WECHAT_CLIENT_SECRET` (and any provider-specific env vars shown above).

## Sign in

```
GET {base_url}/sign-in/social?provider=wechat
```

Callback: `GET {base_url}/callback/wechat`.
