RustAuth is a server-first authentication and authorization framework for Rust. It provides a comprehensive set of features out of the box and includes a plugin ecosystem for advanced functionality — 2FA, passkeys, multi-tenancy, SSO, SCIM, and more — so you can focus on your application instead of reinventing auth infrastructure.

<Callout type="info">
  RustAuth is an independent project, not affiliated with [Better Auth](https://www.better-auth.com). This documentation was adapted from the Better Auth docs (MIT license) to describe RustAuth behavior and APIs.
</Callout>
