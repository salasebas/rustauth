[Agent skills](https://agentskills.io) are portable instruction files (for example `SKILL.md`) that teach your coding agent project conventions, safe patterns, and where to look in the docs.

## Repository skills

The [rustauth](https://github.com/salasebas/rustauth) repository ships maintainer skills under [`.agents/skills/`](https://github.com/salasebas/rustauth/tree/main/.agents/skills) (Rust patterns, testing, database guides). Clone or copy the skill files your agent supports into its skills directory.

Read [AGENTS.md](https://github.com/salasebas/rustauth/blob/main/AGENTS.md) for workspace conventions, verification commands, and upstream parity rules.

## Installing with the skills CLI

If you use the [`skills` CLI](https://www.npmjs.com/package/skills):

```bash title="terminal"
npx skills add salasebas/rustauth --path .agents/skills
```

Your editor or agent loads skills from its configured skills directory (often project-level). After installing, restart the agent or reload skills if your tool requires it.
